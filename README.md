# IT Support Lab Portfolio

Hands-on IT support learner building practical skills in Windows, Active Directory, user management, permissions, and troubleshooting.

## Skills

- Group Policy (GPO)
- File & Folder Permissions
- User Account Troubleshooting
- Windows Server Administration

## Labs
- User and Group Management
- Shared Folder Permissions
- Password Lockout Policy
- Group Policy Restrictions

## Goal
To secure an entry-level IT Support role and build strong troubleshooting and system administration skills.
